Manage Itential Automation Platform

  Find more information at: https://docs.itential.com

Usage:
  iap [command]

Asset Commands:
  clear       Delete all configured instances of a asset
  copy        Copy an asset from one server to another
  create      Create a new asset
  delete      Delete an existing asset
  describe    Display detailed information about a specific asset
  get         Display one or more assets of a specified type

Operator Commands:
  restart     Restart a running application
  start       Start a stopped application
  stop        Stop a running application

Promotion Commands:
  export      Export a asset
  import      Import a asset

Platform Commands:
  api         Send raw API requests to the server
  restart     Restart a running application
  start       Start a stopped application
  stop        Stop a running application
  show        Show status information

Additional Commands:
  version     Print the client version information

Flags:
      --config string    Path to the configuration file
  -h, --help             help for iap
      --output string    Output format (default "human")
      --profile string   Connection profile to use
      --verbose          Enable verbose output

Use "iap [command] --help" for more information about a command.

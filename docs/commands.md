# Commands

The commands provided by `ipctl` are grouped into functional sections.  The
subcommands can be used to manage different aspects of an IAP server.

## Asset Commands

Asset commands are used to manage the different available assets of an IAP
server.

- `copy` - Used to copy an asset directly from one server profile to another
- `clear` - Deletes all instances of a given type of asset
- `create` - Creates a new asset on the server of a given type
- `delete` - Delets a specified asset of a given type from the server
- `describe` - Provides detailed information about a given asset configured on the server
- `get` - Retrieves all assets of a given type from the server and displays them in a tabular format.

Asset commands are designed to make it easy to work with configured asses on
the specified IAP server.   The asset commands are divided in the following
command groups:

- Admin Essentials
- Automation Studio
- Configuration Manager
- Operations Manager
- Lifecycle Manager

# CLI

Itential provides a command line tool for managing Itential Automation Platform
servers.  The command line interface can be used to perform a variety of
operations using the IAP API.

The command line interface is organized into command groups that provide
management of different aspects of IAP.

- **Asset Commands** - Provide CRUD activities for working with IAP documents
- **Service Commands** - Manage the operational (running) state of applications
- **Promotion Commands** - Export and import documents for promoting between
    environments
- **Platform Commands** - Manage the deployed server
- **Snapshot Commands** - Save and load snapshots of asset types
- **Configuration Commands** - Manage the local CLI configuration
